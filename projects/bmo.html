BMO

Designed and built a handheld gaming console inspired by the character BMO. The objective was to develop a self-contained embedded gaming device with a rechargeable battery, display, and user controls, housed in a custom 3D-printed enclosure. The project focused on embedded programming, power management, and product-level mechanical design.

Project Details

Embedded Systems Gaming Console Fusion 360 3D Printing

System Architecture

Processing & Control: Microcontroller-based system handling game logic, display updates, input processing, and a menu-based interface for mode selection.

Display & Input: Integrated screen for graphics rendering and physical buttons for user interaction.

Software Modes: Idle animation mode (BMO face standby), Snake game, Brick destroyer game, and free drawing mode.

Mechanical Design: Enclosure modeled in Fusion 360 and 3D-printed, designed for component fit and button alignment.

Technical Decisions

Designed a modular software structure to support multiple game modes. Implemented a menu-based interface for user navigation. Designed enclosure around component layout rather than retrofitting parts later.

Challenges

Managing limited microcontroller memory for graphics and logic. Optimizing display refresh rate for smooth animation. Designing compact internal layout within enclosure constraints. Ensuring reliable button input without debounce issues.

Results

Built a fully functional handheld gaming console. Implemented multiple interactive software modes. Successfully integrated electronics into a custom 3D-printed enclosure.

Highlights

  • Microcontroller runs multiple game and interaction modes.
  • Menu-based interface enables mode switching on-device.
  • Custom 3D-printed housing integrates display, controls, and power system.
  • Delivered a complete handheld embedded gaming prototype.
