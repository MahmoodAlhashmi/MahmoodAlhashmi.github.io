Donation-Progress Automated Door System

Built an automated door system connected to a donation kiosk, where the door opens proportionally as the fundraising total increases in real time.

Project Details

ESP32RelaysLimit SwitchesAutomationInteractive Installation

System Architecture

The system was divided into a donation kiosk subsystem, a communication layer, and a door mechanism controller. One ESP32 at the kiosk transmitted the fundraising percentage after each successful donation. A separate ESP32 controlled the physical door using relays for motor direction and limit switches for open and closed reference points. On startup, the door performed a calibration routine to establish a reliable internal position reference before proportional movement began.

Technical Decisions

Two separate ESP32 boards were used to isolate kiosk logic from motor control and simplify debugging. A startup calibration routine ensured consistent reference positioning after restart. Limit switches prevented overtravel and reduced mechanical risk. Donation percentage was mapped directly to door travel distance instead of discrete steps, making the mechanism more intuitive.

Challenges

Key challenges included accurate calibration at startup, synchronizing donation updates with door motion without sudden jumps, reducing mechanical stress from frequent position changes, and ensuring safe operation during live public interaction.

Results

The final system was successfully deployed during a live fundraising event, where it updated door position proportionally to donation progress and operated reliably throughout the event.

Highlights

  • A kiosk-side controller transmitted fundraising progress to the door mechanism.
  • A second controller handled motor direction and calibration.
  • Used startup calibration to establish a repeatable position reference.
  • Successfully deployed in a live fundraising event environment.
